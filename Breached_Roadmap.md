| Breachers                               | Custom breaching tools with variable mining speed depending on reinforcement of block and breach/progression balanced durability.                                                                                                                                    | Breachers are required to break blocks inside Landlock claims. Breachers behave like low-durability paxels when used on normal, unreinforced blocks, both inside and outside of claims, but each block mined consumes 4 durability. Iron Breachers have 32 durability. Reinforced blocks have different Breacher durability cost based on reinforcement tier: Wood reinforcement costs 16 Breacher durability. Iron reinforcement costs 64 Breacher durability. | Reinforced Blocks                       | Upgraded regular blocks that protect the base much stronger.                                                                                                                                                                                                         | Players can reinforce blocks in-place using a Reinforcer. The block itself does not change into a new custom block and stays visually the same. The mod stores reinforcement data by block position. To reinforce a block, the player holds a Reinforcer in the main hand and the reinforcement material in the offhand, then right-clicks the target block. Wood Reinforced costs 64 planks and adds 16 Breacher durability cost. Iron Reinforced costs 4 iron blocks and adds 128 Breacher durability cost. Diamond Reinforced costs 2 diamond blocks and adds 256 Breacher durability cost. When a block is reinforced, the material is consumed from the offhand, the original block stays visually unchanged, reinforcement is stored by position, reinforcement persists through server restarts, and reinforcement data is removed if the block is broken. Reinforcement does not create movable reinforced block items. Reinforced blocks use a targeted block outline overlay for visual feedback. When a player is holding either a Reinforcer or any Breacher and looks at a reinforced block, only the single targeted block gets a colored outline. Wood uses a brown/tan outline, Iron uses a gray/white outline, and Diamond uses a cyan/blue outline.                                                           | Working as Intended   | Working as intended                                                                                                                                                           |
