| Reinforced Blocks                    | Upgraded regular blocks that protect the base much stronger.                                                                                                                                                                                                         | Players can reinforce blocks in-place using a Reinforcer. The block itself does not change into a new custom block and stays visually the same. The mod stores reinforcement data by block position. To reinforce a block, the player holds a Reinforcer in the main hand and the reinforcement material in the offhand, then right-clicks the target block. Wood Reinforced costs 8 logs and adds 16 Breacher durability cost. Iron Reinforced costs 2 iron blocks and adds 64 Breacher durability cost. Diamond Reinforced costs 1 diamond block and adds 256 Breacher durability cost. Netherite costs 1 netherite ingot and adds 512 Breacher durability cost. When a block is reinforced, the material is consumed from the offhand, the original block stays visually unchanged, reinforcement is stored by position, reinforcement persists through server restarts, and reinforcement data is removed if the block is broken. Reinforcement does not create movable reinforced block items. Reinforced blocks use a targeted block outline overlay for visual feedback. When a player is holding either a Reinforcer or any Breacher and looks at a reinforced block, only the single targeted block gets a colored outline. Wood uses a brown/tan outline, Iron uses a gray/white outline, and Diamond uses a cyan/blue outline, and netherite is almost a black outline.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     | Working as Intended   | Working as intended                                        | Active          |

Small breached worlds have a cap of 40 minor structures at one time, regular 128.
